# Ueno GraphQL API

## Features

- GraphQL Modules
- TypeGraphQL
- OpenAPI with Swagger
- Jest
- Continous Integration and Deployment

## Getting started

Development

```bash
yarn install
yarn dev
```

Run tests with coverage

```bash
yarn test:coverage
```

## Building and starting

Production

```bash
yarn build
yarn start
```
